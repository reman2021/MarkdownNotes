## 右键菜单
- Win11 改回 Win10 ：`Win + X` -> 终端管理员 -> `reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve` -> 重启 Windows 资源管理器
- Win10 变回 Win11 ：`Win + X` -> 终端管理员 -> `reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve` -> 重启 Windows 资源管理器

## Word
Δ 出现选中后打字重复的问题，如选中文字，按一下c会输入cc

解决方法：
1. 点击“文件”，打开“选项”，点击“加载项”，点击“COM加载项”旁的“转到”。
2. 将里面的加载项都去掉勾选状态，点击“确定”。
